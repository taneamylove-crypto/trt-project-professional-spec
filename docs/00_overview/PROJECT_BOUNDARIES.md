# Project Boundaries

TRT has three independent pillars:

## 1. Draw / Lottery Protocol

Technology layer for creating and running transparent draw mechanics on TON.

## 2. Charity Foundation

Independent social-impact structure funded by treasury allocations, donations or grants.

## 3. Mental Aid Institute

Independent education/support/research structure. Evidence-based psychological support must be clearly separated from spiritual/parapsychological research.

## Important separation

The protocol must not be legally or operationally confused with the foundation or institute.
