# TRT Draw Protocol

The Draw Protocol is the core technology product.

## Main components
- Lottery Factory
- Lottery Contract
- Treasury Contract
- Randomness / Oracle Layer
- Creator Dashboard
- Telegram Mini App
- SDK / API

## Basic flow
1. Creator deploys a draw/lottery contract.
2. Users receive or buy entries.
3. Entries are fixed before draw.
4. Randomness is generated.
5. Winner is determined.
6. Payout is claimed or automatically distributed.
